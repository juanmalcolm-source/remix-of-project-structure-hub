# Requirements — Fractal Kit v1

## v1 Requirements

### Producción Inteligente (PROD)

- [ ] **PROD-01**: Usuario puede generar presupuesto ICAA automáticamente desde el desglose del análisis del guión, con partidas estándar pre-rellenadas por IA
- [ ] **PROD-02**: Usuario puede generar un plan de rodaje optimizado por IA que agrupa escenas por localización, disponibilidad de actores y complejidad técnica
- [ ] **PROD-03**: Usuario puede generar una memoria de producción profesional por IA que incluye sinopsis, ficha técnica, plan de trabajo y justificación artística
- [ ] **PROD-04**: Usuario puede exportar el dossier de producción completo a Excel y PDF con formato profesional
- [ ] **PROD-05**: Usuario puede asignar lugares físicos reales a localizaciones del guión con integración de Google Maps (distancias, tiempos, matriz)
- [ ] **PROD-06**: Sistema aprende de datos reales de rodaje (días reales vs estimados, costes reales vs presupuesto) para mejorar futuras estimaciones

### Distribución y Festivales (DIST)

- [ ] **DIST-01**: Usuario puede consultar una base de datos de festivales españoles (ICAA, clase A, clase B) con filtros por género, duración, temática y fechas
- [ ] **DIST-02**: Usuario puede ver un calendario de convocatorias y deadlines con alertas configurables
- [ ] **DIST-03**: Sistema recomienda festivales usando IA basándose en el análisis DAFO y potencial del guión (matching narrativo-festival)
- [ ] **DIST-04**: Usuario puede hacer tracking de submissions a festivales (estado, fechas, resultados)
- [ ] **DIST-05**: Usuario puede explorar laboratorios de desarrollo y residencias artísticas relevantes para su proyecto
- [ ] **DIST-06**: Usuario puede gestionar tareas relacionadas con convocatorias en un workspace dedicado

### Audiencias (AUD)

- [ ] **AUD-01**: Usuario puede ver buyer personas detallados generados por IA desde el análisis narrativo (demografía, intereses, motivación, canales de alcance)
- [ ] **AUD-02**: Sistema genera estrategia de comunicación por IA conectando análisis narrativo con perfil de audiencia (mensajes clave, canales, timing)
- [ ] **AUD-03**: Usuario puede analizar datos de mercado de audiencias por género cinematográfico y territorio

### Financiación (FIN)

- [ ] **FIN-01**: Usuario puede explorar un mapa de fuentes de financiación (ayudas ICAA, comunidades autónomas, MEDIA, Ibermedia, coproducción, inversión privada)
- [ ] **FIN-02**: Usuario puede simular un plan financiero con diferentes escenarios de financiación
- [ ] **FIN-03**: Sistema calcula intensidad de ayuda e incompatibilidades entre fuentes de financiación según normativa vigente
- [ ] **FIN-04**: Usuario puede ver un timeline de financiación con hitos, deadlines de convocatorias y cashflow proyectado
- [ ] **FIN-05**: Usuario puede configurar territorios de financiación (España, Europa, Iberoamérica) con ayudas específicas de cada territorio

## v2 Requirements (Deferred)

- Integración con Movie Magic / EP Budgeting (formatos de exportación)
- Conectar con APIs de festivales en tiempo real (FilmFreeway, Festhome)
- Multi-idioma (la app está en español; inglés y francés para el futuro)
- Colaboración en tiempo real entre miembros del equipo
- App móvil nativa
- Integración con herramientas de casting (actores/actrices)
- Dashboard de analytics de rendimiento de distribución

## Out of Scope

- **Edición de video** — Fractal Kit cubre pre-producción y distribución, no post-producción
- **CRM de contactos** — Existen herramientas específicas (Hubspot, etc.)
- **Contabilidad/facturación** — Dominio fiscal complejo, mejor integrar con herramientas existentes
- **Streaming/exhibición** — Fuera del ámbito de gestión de producción
- **Marketplace de servicios** — Requeriría modelo de negocio diferente
